# Features & Controls

## Features

- **Instant play** — loads directly in the browser; no install or account
- **Score chasing** — height is the score; every run targets a personal best
- **Quick restarts** — falling costs seconds, not progress screens
- **Desktop and mobile** — keyboard on desktop, touch on mobile

## Controls

| Platform | Input |
|---|---|
| Desktop | Keyboard — the game shows its key bindings on screen |
| Mobile | Touch — tap/swipe as prompted in-game |

:::note
Controls are displayed inside the game and may be refined between releases;
the in-game prompts are always current.
:::

## Requirements

- A current version of Chrome, Edge, Firefox, or Safari
- JavaScript enabled
- No minimum hardware beyond what runs a modern browser comfortably
