# Lisa Climber

**Lisa Climber** is a browser climbing game from Kluth Studios: guide Lisa up
an ever-taller wall, one hold at a time, and see how high you can get before
gravity wins.

**Play it now:** [lisaclimber.kluthstudios.com](https://lisaclimber.kluthstudios.com)

## The objective

Climb. The higher you get, the higher your score — every run is an attempt at
a new personal best. Falls end the run, so each move is a small decision about
risk versus height.

## Why it's fun

- Runs are short, so "one more try" is always cheap.
- The controls take seconds to learn, but clean climbing takes practice.
- No install, no account: the game loads straight into the action.
