# Features & Controls

## Features

- **Classic falling-block gameplay** — clear lines, chase the high score
- **Escalating speed** — difficulty rises as you clear
- **Instant play** — browser-based, no install or account
- **Desktop and mobile** — keyboard or touch

## Controls

| Platform | Input |
|---|---|
| Desktop | Arrow keys to move and rotate (bindings shown in-game) |
| Mobile | On-screen touch controls |

:::note
Controls are displayed inside the game and may be refined between releases;
the in-game prompts are always current.
:::

## Requirements

- A current version of Chrome, Edge, Firefox, or Safari
- JavaScript enabled
