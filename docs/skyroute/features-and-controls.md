# Features & Controls

## Features

- **Flight-and-navigation theme** — plot a route, take to the sky
- **Browser-native** — no install or account
- **Evolving design** — as an Experimental project, new ideas land frequently

## Controls

Skyroute presents its controls on screen. Because the experience is under
active experimentation, the in-experience prompts are the only reference that
is guaranteed current.

## Requirements

- A current version of Chrome, Edge, Firefox, or Safari
- JavaScript enabled

## Known issues

Experimental status means rough edges are expected. Reproducible problems are
still worth reporting — see
[Troubleshooting](/docs/skyroute/troubleshooting) for the checklist first.
