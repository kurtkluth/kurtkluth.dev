SkyRoute Infinite is keyboard-first: left hand on `W`, `A`, `S`, `D` for engine and
rudder, right hand on the arrow keys for pitch and roll. Here is every key in the
game.

## Flying

| Key | What it does |
|---|---|
| `↓` / `↑` | Pitch: `↓` pulls the nose up (stick back), `↑` pushes it down |
| `←` / `→` | Roll left / right |
| `A` / `D` | Rudder, and nosewheel steering while you are on the ground |
| `W` / `S` | Throttle up / down |
| `F` / `Shift+F` | Flaps extend / retract |
| `G` | Landing gear up / down |
| `B` | Brakes |
| `S` (held) | Reverse thrust: jets only, on the runway, with the throttle at idle |

A note on pitch, because it trips everyone up exactly once: the arrows work like a
control stick, not like a camera. Pressing `↓` is pulling the stick back toward
you, which raises the nose. Ten minutes in, it is the only way that feels right.

## Systems and views

| Key | What it does |
|---|---|
| `P` | Autopilot on / off: holds your altitude and follows the route |
| `Tab` | Cycle the autopilot target: SPD, HDG, ALT, V/S |
| `[` / `]` | Adjust the selected autopilot target down / up |
| `,` / `.` | Heading bug minus / plus 10 degrees |
| `N` | Re-arm NAV: track the route laterally |
| `O` | Arm APR: an ILS-style approach that guides you down to the runway |
| `W` / `S` | Target speed up / down while the autopilot is engaged |
| `C` | Chase or cockpit camera |
| `M` | Minimap size / hide |
| `V` | Sound on / off |
| `Esc` | Pause |

:::note

The heading bug is the little marker on your compass that tells the autopilot where
to point in HDG mode. Nudge it around with `,` and `.` in 10-degree steps.

:::

## Cheat sheet by phase of flight

- **Takeoff.** `B` brakes off, `W` full throttle, `↓` to rotate at about 65 kt,
  then `G` gear up and `Shift+F` flaps up.
- **Cruise.** `P` for autopilot; `Tab` plus `[` and `]` to dial in speed, heading,
  altitude, or vertical speed; `M` to check the minimap.
- **Landing.** `F` for flaps, `G` for gear, small pitch and throttle corrections
  all the way down, `B` after touchdown, plus held `S` for reverse thrust in the
  SwiftJet.

New to all of this? Fly the guided first flight in [How to Play](./how-to-play.md),
then go deeper on `P`, `Tab`, `N`, and `O` in the [Autopilot guide](./autopilot.md).
