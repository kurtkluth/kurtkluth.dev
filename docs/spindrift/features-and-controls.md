# Features & Controls

## Features

- **Atmospheric, exploratory design** — no scores, no fail states
- **Browser-native** — no install or account
- **Evolving** — as an Experimental project, it changes shape over time

## Controls

Spindrift presents any needed controls within the experience itself. Because
it is under active experimentation, the in-experience prompts are the only
reference guaranteed to be current.

## Requirements

- A current version of Chrome, Edge, Firefox, or Safari
- JavaScript enabled
- Headphones recommended if the experience includes audio

## Known issues

Experimental status means rough edges are expected. See
[Troubleshooting](/docs/spindrift/troubleshooting) for the usual checklist.
