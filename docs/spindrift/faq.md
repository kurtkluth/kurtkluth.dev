# FAQ

### Do I need to install anything?

No. Spindrift runs entirely in the browser at
[spindrift.kluthstudios.com](https://spindrift.kluthstudios.com). There is nothing to download and no
account to create.

### Does it cost anything?

No — Spindrift is free to play.

### Does it work on mobile?

Spindrift targets current versions of modern browsers on both desktop
and mobile. If something looks wrong on your device, see
[Troubleshooting](/docs/spindrift/troubleshooting).

### Is my progress saved?

Any saved state lives in your own browser (local storage). Clearing site data
for spindrift.kluthstudios.com resets it, and progress does not follow you across
devices or browsers.

### Who made this?

Spindrift is a Kluth Studios project by Kurt Kluth. See the
[About page](/about) for the rest of the collection.
