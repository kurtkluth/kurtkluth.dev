# How to Play

1. Open [spindrift.kluthstudios.com](https://spindrift.kluthstudios.com) in a
   modern browser.
2. Let it load, then follow whatever the experience presents — Spindrift is
   designed to be discovered, not instructed.
3. Take your time. It's an atmosphere, not a race.

:::tip
Spindrift is an evolving experiment: the in-experience presentation always
supersedes these pages. Notable changes are recorded in the
[changelog](/docs/spindrift/changelog).
:::

## If you get stuck

- Reload the page for a clean restart.
- Check [Troubleshooting](/docs/spindrift/troubleshooting) for browser issues.
