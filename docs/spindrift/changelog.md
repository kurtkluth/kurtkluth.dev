## 2026-07-18: Spindrift joins the collection

**Status: Experimental.**

Spindrift joined the Kluth Studios collection on kurtkluth.dev. The live game at
[spindrift.kluthstudios.com](https://spindrift.kluthstudios.com) and this
documentation set went up together. Like every Kluth Studios game, it is free and
runs in any current browser (Chrome, Edge, Firefox, or Safari) with no install
and no account.

Snapshot of the game as documented today:

- **Vector aesthetic.** Pure black screen, glowing white vector outlines,
  monospace type, with the glow toggleable via `G`.
- **Pure arcade loop.** Press `Enter`, clear splitting rocks, fight large and
  small saucers, survive, chase the high score.
- **Drift physics.** Thrust and rotation with no brakes; momentum is the core
  mechanic.
- **Scoring.** Rocks worth 20 / 50 / 100 points by size (big to small), saucers
  worth 200 / 1,000 (large / small), and a bonus ship at 10,000 points.
- **Persistent HI score**, saved in your browser.
- **Hyperspace** (`Shift`) as the risky emergency escape.
- **Inputs.** Keyboard on desktop, touch supported on mobile.
- **Quality-of-life keys.** Pause (`P`), mute (`M`), glow toggle (`G`), and
  fullscreen (`F`).
